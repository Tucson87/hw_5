Список селекторов

  • h1 или любой другой элемент - вызов любого эл-та по названию
  • .class - вызов всех эл-тов с таким именем класса
  • .class 1 class 2 - вызов эл-тов с несколькими указаными классами
  • #id - вызов эл-та с указанным id
  • h1#id - вызов эл-та с указанным именем и id
  • h1.class - вызов эл-та с указаннымименем и классом
  • h1#id + h1.class - вызов эл-та с именем иклассом, который находится возле эл-та с именем и id
  • .class > .class - вызов дочернего эл-та с классом, который находится в родительском классе
  • body h1#id - вызов эл-та по имени и id, который находится в теге body
